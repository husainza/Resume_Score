How to Use

1. Setup

Enter your OpenAI API key in the configuration section above. For deployment, configure GitHub Secrets.

2. Job Description

Enter the job title and description, or upload a file containing the JD.

3. Upload CVs

Drag and drop or select multiple CV files (PDF, DOCX, DOC).

4. Analyze Job Description

Click "Analyze Job Description" to extract requirements and set scoring parameters.

5. Review & Modify

Review the extracted parameters, modify weights, add missing skills, then proceed to Step 2.

6. Upload CVs

Upload CV files (PDF, DOCX, DOC) in Step 2.

7. Analyze CVs

Click "Analyze CVs" to process all candidates using your custom scoring parameters.

8. Review Results

View ranked candidates, filter by score, and export results.
